### Task 4: Create database schema SQL

**`/backend/db/schema.sql`** — Complete SQL from design.md with all 7 tables:
- workers (id, name, phone, platform, city, zone, worker_id, rating, avg_weekly_hours, baseline_weekly_earnings, created_at, updated_at)
- policies (id, policy_number, worker_id FK, start_date, end_date, status, weekly_premium, coverage_amount, created_at, updated_at)
- claims (id, claim_number, worker_id FK, policy_id FK, trigger_type, trigger_timestamp, payout_amount, status, fraud_score, approved_at, paid_at, transaction_id, created_at)
- premium_history (id, worker_id FK, policy_id FK, calculated_premium, base_premium, multiplier, features_json, calculated_at)
- trigger_events (id, trigger_type, zone_id, city, timestamp, duration_hours, intensity_value, source_api, affected_workers_count, event_hash UNIQUE, created_at)
- disruption_zones (id, city, zone_name, lat, lon, flood_risk_score, aqi_risk_score, UNIQUE city+zone_name)
- fraud_flags (id, claim_id FK, flag_type, severity, details_json, flagged_at)

Include all indexes from design.md.
