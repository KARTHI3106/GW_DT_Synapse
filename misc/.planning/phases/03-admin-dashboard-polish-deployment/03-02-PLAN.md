### Task 4: Add page transition animations

<read_first>
- frontend/app/layout.tsx
- frontend/app/globals.css
- frontend/components/ui/card.tsx
</read_first>

<action>
1. Add framer-motion page transition wrapper to layout.tsx:
   ```tsx
   <motion.main initial={{ opacity: 0, y: 8 }} animate={{ opacity: 1, y: 0 }} transition={{ duration: 0.3 }}>
     {children}
   </motion.main>
   ```

2. Add stagger animation to card grids on dashboard and admin pages:
   ```tsx
   const container = { hidden: {}, show: { transition: { staggerChildren: 0.1 } } }
   const item = { hidden: { opacity: 0, y: 20 }, show: { opacity: 1, y: 0 } }
   ```

3. Add hover micro-interactions to cards:
   - Scale 1.02 on hover
   - Border glow (border-indigo-500/30 -> border-indigo-500/60)
   - Transition duration 200ms

4. Add pulse-glow animation to active trigger alerts in conditions panel
</action>

<acceptance_criteria>
- frontend/app/layout.tsx or pages contain `motion.` elements
- frontend/app/globals.css contains animation keyframes (pulse-glow or similar)
- Cards have hover:scale or hover transition
</acceptance_criteria>
