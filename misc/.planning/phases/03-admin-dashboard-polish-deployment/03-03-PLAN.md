### Task 1: Prepare backend for Render deployment

<read_first>
- backend/requirements.txt

- backend/config.py
</read_first>

<action>
1. Create `backend/Procfile`:
   ```
   web: uvicorn main:app --host 0.0.0.0 --port $PORT
   ```

2. Create `backend/render.yaml`:
   ```yaml
   services:
     - type: web
       name: gigshield-api
       runtime: python
       buildCommand: pip install -r requirements.txt
       startCommand: uvicorn main:app --host 0.0.0.0 --port $PORT
       envVars:
         - key: SUPABASE_URL
           sync: false
         - key: SUPABASE_KEY
           sync: false
         - key: OPENWEATHERMAP_API_KEY
           sync: false
         - key: WAQI_TOKEN
           sync: false
         - key: RAZORPAY_KEY_ID
           sync: false
         - key: RAZORPAY_KEY_SECRET
           sync: false
   ```

3. Ensure `requirements.txt` has all dependencies (including razorpay if added)

   ```python

   ```

5. Add health check endpoint if missing:
   ```python

   async def health():
       return {"status": "healthy", "service": "gigshield-api"}
   ```

### Task 2: Prepare frontend for Vercel deployment

<read_first>
- frontend/next.config.js
- frontend/package.json
- frontend/.env.local
</read_first>

<action>
1. Update `frontend/next.config.js` to add API proxy rewrite for production:
   ```javascript
   module.exports = {
     async rewrites() {
       return [
         {
           source: '/api/v1/:path*',
           destination: `${process.env.NEXT_PUBLIC_API_URL}/api/v1/:path*`,
         },
       ]
     },
     images: {
       domains: [],
     },
   }
   ```

2. Create `vercel.json` in project root (optional, for monorepo):
   ```json
   {
     "buildCommand": "cd frontend && npm run build",
     "outputDirectory": "frontend/.next",
     "installCommand": "cd frontend && npm install",
     "framework": "nextjs",
     "rewrites": [
       { "source": "/api/v1/:path*", "destination": "https://gigshield-api.onrender.com/api/v1/:path*" }
     ]
   }
   ```
   
3. Ensure `frontend/.env.local` template documents required env vars:
   ```
   NEXT_PUBLIC_API_URL=https://gigshield-api.onrender.com
   ```

4. Verify build passes: `cd frontend && npm run build`
</action>

<acceptance_criteria>
- frontend/next.config.js contains rewrites for API proxy
- vercel.json exists in project root
- frontend/.env.local or .env.example documents NEXT_PUBLIC_API_URL
</acceptance_criteria>
